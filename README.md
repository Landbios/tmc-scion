# S.C.I.O.N. (Secure Communication Interface for Operative Network)

A web application for TMC chatrooms and character management.


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
